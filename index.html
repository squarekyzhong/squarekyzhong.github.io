Fangtian Zhong

Currently I am a final year PhD student in The Department of Computer Science at George Washington University, Washington, DC, USA, advised by Prof. Xiuzhen Cheng. Before that, I received his B.E. degree from Northeast Normal University where I fell in love with computer security.

My principal research interests lie in software security, system security, and machine learning applications in security, and adversarial machine learning. I mainly focus on three research topics as follows: 1) Binary Analysis 2) Machine Learning for Cybersecurity Problems 3) Automatic Binary Parsing.

  • MalFox: GAN-based Adversarial Malware Example Attacks

    • Machine learning has been broadly investigated and showed great merits in various applications. Nevertheless, with its rapid development, security issues have raised serious alarms in recent years. A variety of adversarial examples used in different areas thrive. However, there have been no successful attacks against the collection of practical antivirus products and online scan engines, e.g. VirusTotal, due to the difficulty of generating practical adversarial malware examples that retain original malware functionalities unchanged. We develop a convolutional generative adversarial network-based (Conv-GAN) generation framework titled MalFox, which interacts with practical antivirus products and scan engines in VirusTotal to generate powerful adversarial malware examples by leveraging the bugs of these entities that cannot check for kernel-level modification and concealment of malware.

  • Malware-on-the-Brain: Illuminating Malware Bytes with Images

    • Malware is a piece of software that was written with the intent of doing harm to data, devices, or people. Since a number of new malware variants can be generated by reusing codes, malware attacks can be easily launched and thus become common in recent years, incurring huge losses in businesses, governments, financial institutes, health providers, etc. To defeat these attacks, malware classification is employed, which plays an essential role in anti-virus products. However, existing works that employ either static analysis or dynamic analysis have major weaknesses in complicated reverse engineering and time-consuming tasks to distinguish different categories of adversarial malware examples and malware samples. This drives us to provide a highly efficient classification framework with acceptable accuracy. To achieve this goal, we propose a visualized malware classification framework called VisMal, which converts malware samples into images and then applies a contrast-limited adaptive histogram equalization algorithm to enhance the similarity between malware image regions in the same family.

  • User Response Prediction by Deep Cross Product Network with An Attention Scheme

    • User response prediction, which aims to predict the probability that a user will provide a predefined positive response in a given context such as clicking on an ad or purchasing an item, is crucial to many industrial applications such as online advertising, recommender systems, and search ranking. For these tasks and many other machine learning tasks, an indispensable part of success is feature engineering, where cross features are a significant type of feature transformation. However, due to the high dimensionality and super sparsity of the data collected in these tasks, handcrafting cross features is inevitably time expensive. Existing methods can be hindered by not learning sufficient cross features due to model architecture limitations or modeling all high-order feature interactions with equal weights. To fill this gap, we propose a novel architecture deep cross-product network with an attention scheme, which allows practitioners to perform an in-depth analysis of user behaviors.
